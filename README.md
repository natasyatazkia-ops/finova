# 💜 Finova

Finova adalah aplikasi **Personal Finance Tracker** yang dibuat menggunakan **React JS** dan **Tailwind CSS**.

## Fitur
- ✅ Dashboard (Saldo, Income, Expense)
- ✅ Tambah Transaksi
- ✅ Edit Transaksi
- ✅ Hapus Transaksi
- ✅ Search
- ✅ Filter
- ✅ LocalStorage

## Anggota Kelompok
1. Natasya Tazkia
2. Nama Teman

## Teknologi
- React JS
- Vite
- Tailwind CSS
- JavaScript
